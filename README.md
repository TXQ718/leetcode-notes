# leetcode-notes

LeetCode 刷题学习笔记，使用 Python 解题，记录解题思路、代码实现、易错点和复杂度分析。

## 目录结构

```
├── hot100/          # 分类题目笔记（Hot 100）
├── Daily/           # 每日刷题总结
├── template/        # 笔记模板
│   ├── problem-template.md   # 单题笔记模板
│   └── daily-template.md     # 每日总结模板
└── README.md
```

## 笔记格式

### 题目笔记（[hot100/](hot100/)）

每道题包含：题目信息与描述 → 解题思路 → Python 代码实现 → 代码解释 → 语法总结 → 易错点与边界情况 → 复杂度分析。

文件命名：`{题号}-{英文标题}.md`，如 `0001-two-sum.md`。

### 每日总结（[Daily/](Daily/)）

每天刷题后回顾当日内容，包括：完成题目表 → 题目复盘 → Python 语法收获 → 与 C++ 对比 → 易错点汇总 → 新掌握的模板 → 明日计划。

文件命名：`{YYYY-MM-DD}-daily-summary.md`。

## 使用方式

1. 从 [template/problem-template.md](template/problem-template.md) 复制模板，创建新题目笔记
2. 从 [template/daily-template.md](template/daily-template.md) 复制模板，记录每日总结
3. 题目笔记放入对应分类目录，每日总结放入 `Daily/`
